<p><strong>Parametric Literacy Tool — Enhanced Educational Edition v1.0</strong></p>

<p>A web-based educational application designed to teach <strong>parametric literacy</strong> in digital image processing — the ability to understand and articulate visual outcomes as systematic configurations of adjustable parameters rather than vague stylistic descriptions.</p>

<p>Instead of relying on imprecise instructions like "make it cinematic," students learn to decompose visual intentions into explicit numerical specifications (e.g., contrast +15, temperature −5, vignette 30%). The tool implements <strong>scaffolded decomposition</strong>: each parameter represents a discrete design decision with immediate visual feedback.</p>

<p><strong>Key features:</strong></p>
<ul>
<li>Real-time live preview with non-destructive parametric editing</li>
<li>Automatic AI prompt generation from parameter configurations</li>
<li>Parametric breakdown teaching panel with pedagogical annotations</li>
<li>Natural language summary of current parameter state</li>
<li>Context-aware smart tips and parameter complexity meter</li>
<li>Vague vs. parametric language comparison</li>
<li>Four parameter categories: Basic Adjustments, Color Manipulation, Advanced Effects, Overlays &amp; Composition</li>
</ul>

<p><strong>This package contains:</strong></p>
<ul>
<li><code>index.html</code> — Landing page (Educational Applications Management System)</li>
<li><code>enhanced_parametric_app.html</code> — Main interactive application</li>
<li><code>parametric-literacy-manual-fixed.html</code> — Educational Manual for Digital Image Processing (v1.0)</li>
<li><code>README.md</code> — Documentation and usage instructions</li>
</ul>

<p>Open any HTML file in a modern web browser. No installation or server required.</p>
