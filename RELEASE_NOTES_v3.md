# Parametric Literacy Tool v3.0.0 Release Notes

Version 3.0.0 is the public release of the enhanced Parametric Literacy Tool.

## Main Improvements

- Added a built-in AI-generated demo landscape image for immediate testing.
- Added parameter presets and reset-by-group controls.
- Added undo/redo support for parameter experimentation.
- Added before/after split view for visual comparison.
- Added histogram metrics:
  - mean luminance,
  - clipped shadows percentage,
  - clipped highlights percentage,
  - average saturation.
- Added warning feedback for extreme values and clipping risk.
- Added parameter search.
- Added JSON import to complement JSON export.
- Added point zoom inspection for local detail, noise, blur, sharpening, and clipping.
- Added local non-AI educational feedback when no API key is available.
- Added AI image-analysis privacy confirmation.
- Added simple/technical prompt comparison.
- Improved blur and sharpening implementations.
- Made grain/noise reproducible using a seed in metadata.
- Updated the technical manual and index page with new screenshots.
- Added DOI-based scientific tooltip references.
