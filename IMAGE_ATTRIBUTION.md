This prototype uses an AI-generated landscape image as the built-in educational demo image.

File:

`parametric_manual_assets/demo-landscape.png`

Description:

AI-generated photorealistic landscape with sea, mountains, sunset light, clouds, shadows, highlights, and warm color gradients. It is included for educational image-editing demonstration and histogram analysis.

Suggested credit line:

AI-generated demo image created with ChatGPT on 18 May 2026 for educational software demonstration purposes.

## Usage Note

The demo image is not part of the software source code license. It is included as a visual test asset so users can immediately explore brightness, contrast, gamma, saturation, temperature, clipping, histogram metrics, and zoom inspection without uploading a personal image.

The image should be described as an AI-generated demonstration asset. It should not be presented as photographic evidence or as a real documented location.
