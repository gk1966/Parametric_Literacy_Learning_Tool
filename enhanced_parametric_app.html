What is Parametric Literacy?

Parametric literacy is the ability to understand and articulate visual outcomes as systematic configurations of adjustable parameters rather than vague stylistic descriptions.

Visualizing Image Data

Histogram Analysis: The graph represents tonal distribution. It is divided into Shadows (left), Midtones (center), and Highlights (right).
Reference base: The help popups cite selected sources on digital image processing, histogram/gamma correction, color processing, and gamut clipping. Use the DOI links inside each popup for more information.
Bibliography: Sinecen (2016), DOI 10.5772/63028; Peli (1992), DOI 10.1117/12.59989; Yang and Kwok (2003), DOI 10.1117/1.1544479; Subramani and Veluchamy (2020), DOI 10.1002/col.22502; Kadmin (2022), DOI 10.24996/ijs.2022.63.1.30; Oduncu et al. (2004), DOI 10.1177/1534734604268842.
Clipping / Data Loss: When you adjust extremely, pixels push off the edges (below 0 or above 255), destroying detail. Use "Show Clipping" under the canvas to visualize lost shadows (Blue) and highlights (Red).

API setup, free access, and privacy notes

How to get a Gemini API key: open Google AI Studio, create or select an API key, and paste it above. Availability, rate limits, and pricing can change; always check the official Gemini API pricing and rate limits pages.

Privacy: core image editing runs locally in the browser. The current canvas image and active parameter context are sent to Gemini only when you click Analyze My Image. Metadata JSON exports do not include image pixels or API keys.

Other providers to explore: Hugging Face offers inference providers with small monthly credits for experimentation, while Cloudinary offers a free plan for image transformation APIs. These providers are not currently integrated in this tool.
